# Multi-agent 智能客服 - 完整项目说明

## 📋 项目概述

基于 **LangGraph** 的多智能体客服系统，集成了意图路由、问答机器人、工单管理和人工客服转接功能。采用前后端分离架构，适合作为简历项目或进行二次开发扩展。

## 🎯 核心功能

### 1. 智能意图识别
- **FAQ 咨询**：自动回答常见问题（退货、退款、配送等）
- **工单处理**：查询、创建、更新工单
- **人工转接**：识别用户需求，转接人工客服

### 2. 多轮对话支持
- 基于 `MemorySaver` 的会话持久化
- 同一 `session_id` 保持上下文连贯性
- 支持连续对话和追问

### 3. MCP 工具集成
- **知识库服务**：内置 FAQ 库，关键词匹配
- **工单服务**：内存存储，支持 CRUD 操作
- 可扩展更多自定义工具

## 🏗️ 技术架构

### 后端技术栈
- **编排框架**：LangGraph（状态图、条件边、节点）
- **大模型**：阿里云千问（qwen-max）
- **MCP 协议**：FastMCP + langchain-mcp-adapters
- **API 服务**：FastAPI + uvicorn
- **环境变量**：python-dotenv

### 前端技术栈
- **纯原生**：HTML5 + CSS3 + JavaScript
- **零依赖**：无需任何框架和构建工具
- **现代 UI**：渐变主题、流畅动画、响应式布局

## 🚀 快速开始

### 方式一：手动启动

**1. 安装依赖**
```bash
cd "c:\Resume Projects\Multi-agent"

```

**2. 配置环境变量**
```bash

# 编辑 .env 文件，填入有效的 DashScope API Key
# 获取地址：https://dashscope.console.aliyun.com/apikey
```

**3. 启动后端服务**
```bash

uvicorn app.main:app --reload
```

后端服务运行在：http://localhost:8000

**4. 打开前端页面（可选）**

- **直接打开**：双击 `frontend/index.html`
- **HTTP 服务器**：
  ```bash
  cd frontend
  python -m http.server 8080
  ```
  然后访问：http://localhost:8080

## 🔧 配置说明

### 环境变量（.env）

```bash
# 阿里云 DashScope API Key（必填）
DASHSCOPE_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

# 可选：MCP 启动命令（默认使用 python -m 方式）
# MCP_KNOWLEDGE_COMMAND=python
# MCP_KNOWLEDGE_ARGS=-m,mcp_servers.knowledge_server
```

### 模型配置

当前使用 `qwen-max` 模型，可在 `app/graph.py` 中修改：

```python
def _get_llm():
    return ChatTongyi(
        model="qwen-max",  # 可改为 qwen-plus, qwen-turbo 等
        dashscope_api_key=os.environ.get("DASHSCOPE_API_KEY", ""),
        streaming=False,
    )
```

**可选模型对比：**
| 模型 | 特点 | 适用场景 |
|------|------|----------|
| qwen-max | 效果最好，有免费额度 | 生产环境、高质量要求 |
| qwen-plus | 中等价格，效果好 | 平衡性能与成本 |
| qwen-turbo | 经济实惠，响应快 | 快速测试、低成本 |

## ⚠️ 注意事项

1. **必须从项目根目录启动**：确保 `python -m mcp_servers.*` 能正确导入
